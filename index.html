01

Purpose

Script 3 is the core calculation engine of the GHG BEX platform. It takes raw event activity data and matches each row against the best available emission factor from dmhistory, then computes CO₂ equivalent emissions.

ENV = "bex"  # BernExpo → reads fcinbex, writes fcoutbex
ENV = "se"   # Sustainable Events → reads fcinse, writes fcoutse

High-level flow

fcinbex / fcinse
→
Exclusions
→
Normalize keys
→
Tiered match
→
Calculate CO₂
→
fcoutbex / fcoutse

Key design principle: Every input row receives exactly one output row. Orphans appear with null CO₂ — visible in QA for follow-up.

05

Key normalization

# Applied to: ActivitySource, ActivityEN, ActivityID, Context
lowercase · trim · non-breaking space → regular · collapse multiple spaces · ContextDate → int

The double-space bug: "chair: without armrest black" → after normalization becomes identical to the 1-space version → C3 duplicate. Clean source data in SharePoint.

CO₂ calculation

QtyTot = Qty × AssumptionValue
EmissionsKgCO2eq = QtyTot × EmissionFactor
entrecôte — 125 portions → 727.6 kgCO₂e
125 portions × 0.2 kg= 25 kg
25 kg × 29.1043= 727.6 kgCO₂e
security — 2 unitday → 12 kgCO₂e
2 unitday × 1 day/unitday= 2 days
2 days × 6 kgCO₂e/day= 12 kgCO₂e

Assumption guide

NEW
QtyUnit × AssumptionUnit = EF unit
FieldRule
AssumptionValueAlways fill. Minimum = 1. Null = no CO₂.
AssumptionUnitDenominator must cancel QtyUnit. Empty if already in EF unit.
AssumptionNoteAlways fill for traceability. Not used in calculation.
